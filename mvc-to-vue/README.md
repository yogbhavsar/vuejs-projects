# Migrate an ASP .NET application page-by-page to VueJS 3 with Typescript

This repo deals with the sample solutions generated by command line tools of .NET and VueJS 3 with npm. It has changes to migrate the front-end of the Privacy page from the sample MVC application to Vue JS 3 components. 

## How to run
Simply build and run the `MvcToVue` project! Note that the project uses .NET 10 and the Vue solution uses node v24.11.0.

## Step by Step
Checkout [my dev.to post](https://dev.to/yogesh_bhavsar/how-to-convert-an-asp-net-mvc-application-to-vue-js-3-with-typescript-page-by-page-2l84-temp-slug-1021057) for detailed steps on how did this code got to the final stage.

## Disclaimer
The base code is as per the generated code by commands as described in the above blog post, as of 16 April, 2026. It may change going forward.
